# Luxury Watch E-Commerce Landing Page

A responsive luxury watch store landing page built with React and Tailwind CSS.

## Getting Started
```bash
git clone https://github.com/rajrasane/intern-luxury-watch-project.git
cd intern-luxury-watch-project
npm install
npm run dev
```

## Features

- Fully responsive design (Desktop, Tablet, Mobile)
- Light & Dark theme toggle
- Modern luxury aesthetic
- Built with React + Tailwind CSS
