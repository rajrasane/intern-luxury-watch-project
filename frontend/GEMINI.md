## Design Specifications

### Breakpoints & Grids
- **Desktop**: 1440px width, 12-column grid, content max 1200px centered
- **Tablet**: 768px width, 8-column grid
- **Mobile**: 375px width, 4-column grid


### Design Style
- Luxury/premium aesthetic
- Generous whitespace
- Large, high-quality product images
- Minimal UI elements
- Smooth transitions and interactions

## Required Sections (in order)

### 1. Header
- Logo (left)
- Navigation menu (center): Home, Collections, About, Contact
- Right side icons: Search, Cart, Light/Dark theme toggle
- Should be sticky/fixed on scroll

### 2. Hero Section
- Large featured luxury watch image
- Headline/tagline
- "Shop Now" CTA button
- Full-width or prominent positioning

### 3. Featured Collections
- 3 collection cards: Classic, Sport, Limited Edition
- Each card: Image, collection name, "Explore" link
- Layout: 3 columns (desktop), 2 columns (tablet), 1 column (mobile)

### 4. Bestsellers/New Arrivals
- Grid of watch products (4-8 watches)
- Each product: Image, name, price, hover state
- Layout: 4 columns (desktop), 2 columns (tablet), 1 column (mobile)

### 5. Brand Story Section
- "OUR STORY" label
- Heading and descriptive text about heritage/craftsmanship
- Large supporting image (50/50 split with text)
